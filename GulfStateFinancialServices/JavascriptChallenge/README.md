# Tag Validator

## Pre-requisites
- Please download and install `node` from [here](https://nodejs.org/en/)
- Execute `node -v` to make sure it is in $PATH and installation is successfull
- Execute `npm` to install the `typescript` cli

    ```sh 
    npm install -g typescript
    ```
    
## Running the validator
- `cd` into the directory containing `tagValidator.ts`
- Execute `tsc tagValidator.ts` <-- Transpiling step
- Execute the JS file that was created `node tagValidator.js`
